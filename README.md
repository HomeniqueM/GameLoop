# GameLoop
<p>Create a game loop from scratch using the C programming language and SDL</p><br>


## Dependencies
    

- Debian  

GCC compiler

    sudo apt install build-esssential
SDL 

    sudo apt install libsdl2-dev

- Arch 

GCC compiler

    sudo pacman -S gcc

SDL

    sudo pacman -S libidl2


## Executed
### Run from root folder

Bluid 

    make

Run 

    makr run
